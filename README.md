# IEX Cloud (iex-cloud) - SUNSET 2024-08-31

IEX Cloud was a cloud-based financial data platform from IEX Group offering market data, fundamentals, news, options, and reference APIs at `https://cloud.iexapis.com/v1`. The service reached end-of-life on **August 31, 2024**. This entry is preserved as a historical reference for migration patterns; endpoints are no longer reachable.
